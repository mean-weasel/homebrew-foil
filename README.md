# Homebrew Foil

Homebrew tap for [Foil](https://github.com/mean-weasel/foil), a macOS menu bar speech-to-text app with cloud and local transcription providers.

Product site: <https://mean-weasel.github.io/foil/>

```sh
brew tap mean-weasel/foil https://github.com/mean-weasel/homebrew-foil
brew install --cask foil
```

Prefer a manual install? Download the latest signed DMG from [Foil releases](https://github.com/mean-weasel/foil/releases/latest).
